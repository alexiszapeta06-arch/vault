Sin cuenta
1
Habilita Depuración USB
Ajustes → Acerca del teléfono → toca "Número de compilación" 7 veces → Opciones de desarrollador → Depuración USB
2
Conecta tu teléfono por USB
Usa un cable USB que transmita datos (no solo carga)
3
Toca "Instalar Vault"
El navegador pedirá acceso al dispositivo USB
4
Acepta "Confiar en esta PC"
Aparece una vez en la pantalla de tu teléfono
5
Instalación automática
Termux + AuroraOS + Vault se instalan solos
6
¡Listo! Vault está en tu teléfono
Desconecta USB y abre Vault
⚠ Tu navegador no soporta WebUSB. Usa Chrome o Edge en escritorio, o usa la pestaña "Manual" para instalar por terminal.
Paso 1
Instala ADB en tu PC
# macOS:
brew install android-platform-tools

# Windows: descarga Platform Tools de developer.android.com

# Linux:
sudo apt install adb
Paso 2
Conecta y verifica
adb devices
Paso 3
Instala Termux
adb install termux.apk
Paso 4
Instala Vault desde Termux
pkg install -y curl && curl -fsSL https://alexiszapeta06-arch.github.io/vault/install.sh | bash
Requisitos
  • Android 7.0+ (SDK 24+)
  • Cable USB con transferencia de datos
  • Chrome o Edge (para instalación automática)
  • ~300 MB de espacio libre
  • Depuración USB habilitada
Vault es software experimental en fase beta.
